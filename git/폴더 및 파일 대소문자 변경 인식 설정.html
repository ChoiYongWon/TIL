# 폴더 및 파일 대소문자 변경 인식 설정

git은 기본적으로 파일 및 폴더명의 대소문자를 구분하지 않는다.

Apple 폴더를 apple으로 변경해도 git은 인식하지 못한다.

아래 명령어를 입력하면, 대소문자를 구분한다.

git config core.ignorecase false
