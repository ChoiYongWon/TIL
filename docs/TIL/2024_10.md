### 하나의 로컬 환경에서 여러 Github 계정 관리하기

gh 패키지로 로그인 이후 따로 ~/.config/gh/host.yml을 분리 이후 gh alias로 간단하게 계정 스위칭 설정

추가적으로 해당 디렉토리 내에서 동작하게 끔 하려면 autoenv 사용
